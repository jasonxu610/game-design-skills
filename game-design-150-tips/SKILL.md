# 爆款方法论：游戏设计的3大核心与150个技巧

> 本技能基于曾宣凯所著《爆款方法论》提炼，提供游戏设计的系统方法论框架与150个实操技巧。

## 核心理念

**把握不变**：游戏行业变化极快，但底层规律不变——玩家永远追求更高的**娱乐性价比**。

**核心公式**：
```
娱乐性价比 = (游戏体验 / 游戏成本) × 玩法新颖系数
```

**心法 > 招式**：不抄表面形式，而是理解爆款成功的底层逻辑（心法→精髓→灵感→新品）。

**渐进式创新**：在成功基础上逐步加入创新元素，可进可退，用AB测试验证。

## 工作模式

根据你的需求选择对应模式：

### 模式A：从零设计
当你需要**从头设计一款新游戏**时：
1. 明确目标用户与市场定位 → 技巧78, 72-73
2. 选择制作模式（换皮/精进/融合/情境）→ 技巧82-85
3. 设计核心玩法循环 → 技巧1-3, 10-13
4. 构建心理动机模型 → 技巧4-9
5. 推演战斗/交互系统 → 技巧18-21, 31-36
6. 设计深度与精通路径 → 技巧41, 49-56
7. 验证用户需求 → 技巧100-108
8. 规划版本迭代 → 技巧96-99

### 模式B：专项咨询
当你有**特定设计问题**时，查阅技巧速查表定位答案：
- "游戏不够好玩" → 技巧4-9（动机）, 技巧55-56（峰值体验）
- "玩家流失严重" → 技巧87-90（留存）, 技巧111（长期留存）
- "操作太复杂" → 技巧22-27（操作设计）, 技巧49（更简单原则）
- "付费数据差" → 技巧129-130（付费动力）, 技巧95（商业成本）
- "PVP平衡问题" → 技巧115-116（PVP设计）, 技巧43（反制原则）
- "缺乏创新" → 技巧63-77（创新方法论）, 技巧82（整合性创新）
- "社交系统薄弱" → 技巧141（社交设计）, 技巧60（玩家分类）

### 模式C：诊断修复
当你的游戏**已上线但数据不佳**时：
1. 定位问题：留存？付费？DAU？→ 技巧86-90
2. 分析用户行为数据 → 技巧93, 99, 110
3. 诊断核心原因 → 技巧117（破解原因）, 技巧114（机制阻碍）
4. 制定修复方案 → 技巧96-98（版本更新）
5. 验证效果 → 技巧86（测试批次）, 技巧102（可证伪原则）

### 模式D：评审优化
当你需要**评审一个设计方案**时：
1. 用技巧1（主体视角流程）模拟自检
2. 用技巧3（6级设计优先原则）检查设计层级
3. 用技巧11（吸引力公式）评估吸引力
4. 用技巧25（极限策略校验）自我否定
5. 用技巧55-56（峰值体验）检查体验曲线
6. 用技巧150（问题解决流程）处理发现的问题

## 关键模型与公式

1. **娱乐性价比公式**：`(游戏体验/游戏成本) × 玩法新颖系数`
2. **主体视角三角**：剖析者（机制-运行-体验）/ 玩家（表现-体验-运行）/ 旁观者（表现-体验）
3. **8大心理动机**：成就、探索、社交、竞争、收集、创造、沉浸、掌控
4. **多巴胺产出模型**：新鲜感 > 变化 > 不确定性 > 奖励
5. **6级设计优先原则**：灵魂 > 机制 > 运行 > 表现 > 体验 > 细节
6. **五层系统模型**：核心 > 外延 > 辅助 > 装饰 > 冗余
7. **ERG需求模型**：存在需求 / 关系需求 / 成长需求
8. **卡诺模型**：必备功能 / 期望功能 / 惊喜功能
9. **守破离流程**：守（学习模仿）→ 破（突破规则）→ 离（自成一派）
10. **峰值体验模型**：峰值 = 正面体验巅峰 - 负面体验谷底
