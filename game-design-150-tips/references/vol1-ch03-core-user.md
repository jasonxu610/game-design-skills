# 第3章：把握你的核心用户（技巧10-13）

## 核心概念

理解核心用户是设计好玩法的前提。需要从多维度分析游戏的吸引力构成。

## 技巧10：深度优先原则与真理边界

- **深度优先**：优先把核心玩法做深，而非铺广度
- 深度产生的乐趣远大于广度
- 每个"真理"都有其适用边界，不要过度泛化
- 一个精妙的核心循环 > 十个浅薄的系统

## 技巧11：游戏吸引力公式与解构要点

游戏吸引力 = 核心体验质量 × 目标用户匹配度 × 表现力 × 新颖度

解构要点：
1. 核心体验是否足够强烈
2. 是否精准匹配目标用户的动机
3. 表现力是否让玩家快速理解玩法
4. 与市场同类产品的差异化程度

## 技巧12：游戏多元智能模型与数理逻辑

游戏可调动的多种智能类型（基于加德纳多元智能理论）：
- **逻辑数学智能**：策略、数值计算、概率判断
- **空间视觉智能**：地图探索、建筑、射击瞄准
- **身体动觉智能**：操作技巧、反应速度、节奏感
- **音乐节奏智能**：音游、节奏类玩法
- **人际交往智能**：社交、团队合作、谈判
- **自我认知智能**：角色扮演、剧情选择、道德抉择
- **自然观察智能**：收集、分类、生态系统

**应用**：分析你的游戏调动了哪些智能，是否与目标用户的智能偏好匹配。

## 技巧13：游戏进程对比模型与多维视角

从多个维度对比分析游戏进程：
- 时间维度：新手期→成长期→成熟期→衰退期
- 内容维度：核心循环→外围系统→社交系统→运营内容
- 体验维度：新鲜感→掌控感→成就感→归属感
- 对比不同游戏在各维度的处理方式，找到优化方向
