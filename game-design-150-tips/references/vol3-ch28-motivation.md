# 第28章：用户动机与行为（技巧104-108 扩展）

## 核心概念
用户行为是用户动机的外在表现。理解动机-行为的关系是调控游戏的关键。

### 动机→行为的转化链
动机（内在驱动）→ 追求（外在目标）→ 行为（实际操作）→ 反馈（结果感知）→ 新动机

### 关键原则
1. 观察行为推断动机，而非直接询问
2. 同一行为可能源于不同动机
3. 同一动机可能导致不同行为
4. 设计应同时考虑动机和行为两个层面

### 动机类型
- **内在动机**：乐趣、好奇、掌控感、创造欲
- **外在动机**：奖励、排名、社交认可、成就展示
- **社交动机**：归属感、竞争、合作、展示

### 实操建议
1. 用数据埋点记录玩家行为
2. 从行为模式推断主要动机
3. 针对主要动机优化游戏设计
4. 验证优化后行为是否改善
