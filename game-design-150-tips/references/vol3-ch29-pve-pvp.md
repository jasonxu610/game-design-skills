# 第29-30章：PVE/PVP追求层级（技巧109-116）

## PVE游戏的追求层级（第29章）

### 技巧109：需求的跨级追求方法与整体串联
- PVE玩家的需求层级：生存→成长→探索→创造→意义
- 设计需串联各层级，形成完整的追求链
- 让玩家自然地从低层级升级到高层级

### 技巧110：4元分析模型与双重标准
四维度分析：
1. 玩家想要什么（主观需求）
2. 玩家需要什么（客观需求）
3. 玩家做了什么（实际行为）
4. 玩家感受如何（体验反馈）
- **双重标准**：主观和客观需求可能矛盾

### 技巧111-113：留存与转化
- 技巧111：长期留存的成本效益分析
- 技巧112：锁定核心需求让玩家沉浸
- 技巧113：三级转化标准（认知→体验→付费）

## PVP游戏的用户兼顾（第30章）

### 技巧114：机制阻碍理论与定性分析
- 识别阻碍用户留存的机制问题
- PVP中常见阻碍：被碾压、匹配不公、外挂
- 定性分析找出最严重的阻碍因素

### 技巧115：PVP玩家行为模型与利己主义
- PVP玩家本质是**利己的**
- 玩家追求：赢→展示→获得认可
- 设计需满足胜者的成就感，同时不过度打击败者
- 败者需要"下次能赢"的希望

### 技巧116：PVP玩法兼顾的流程与流派调整
1. 识别不同PVP流派（竞技/休闲/社交/策略）
2. 分析各流派的核心需求差异
3. 设计兼顾方案（分模式/分段位/分玩法）
4. 用流派调整平衡各方体验
