# 游戏设计的 236 个技巧

> 副标题：游戏机制、关卡设计和镜头窍门
> 从游戏设计者和玩家的双重角度出发，以《战神》《猎天使魔女》《塞尔达传说》等大量经典游戏为例，探讨如何让 3D 游戏更加有趣。

## 五大模块

### 一、玩家角色（Player Character）
设计玩家操控角色的核心技巧：
- **移动系统**：行走、奔跑、跳跃、攀爬的操作手感
- **动作系统**：攻击、防御、闪避、连招的设计原则
- **状态系统**：HP、体力、buff/debuff 的反馈设计
- **成长系统**：能力解锁、技能树、装备影响
- **操控感**：输入响应延迟、动画过渡、惯性物理

### 二、敌人角色（Enemy Character）
设计有趣且有挑战性的敌人：
- **AI 行为模式**：巡逻、追踪、包围、撤退
- **攻击模式**：远程/近战、组合攻击、可预判的攻击前摇
- **难度曲线**：敌人类型递进、数值成长、新机制引入节奏
- **Boss 设计**：多阶段战斗、弱点暴露、仪式感
- **敌人反馈**：受击反应、死亡动画、击杀满足感

### 三、关卡设计（Level Design）
构建引人入胜的游戏空间：
- **空间布局**：引导动线、探索奖励、秘密区域
- **节奏控制**：战斗-休息-探索的交替节奏
- **环境叙事**：通过场景讲故事
- **难度递进**：教学关→挑战关→Boss 关的渐进设计
- **可重玩性**：支线路径、隐藏要素、速通设计

### 四、碰撞检测（Collision Detection）
游戏物理与交互的技术设计：
- **碰撞体设计**：hitbox/hurtbox 的大小与层级
- **判定时机**：攻击判定帧、无敌帧、取消窗口
- **空间交互**：推箱子、开关、可破坏物
- **物理反馈**：击退、击飞、弹射的力度设计
- **容错设计**：平台边缘容差（coyote time）、输入缓冲

### 五、镜头控制（Camera）
3D 游戏镜头的设计窍门：
- **视角类型**：第三人称跟随、固定镜头、半固定镜头
- **镜头运动**：平滑跟随、预测移动、碰撞回避
- **战斗镜头**：锁定模式、多目标切换、Boss 特写
- **叙事镜头**：过场动画、QTE 镜头、环境展示
- **玩家控制**：镜头灵敏度、自动回正、FOV 调节

## 工作模式

### 模式 1：从零设计
触发："我要设计一个 3D 动作游戏"
→ 按五大模块顺序走完设计流程

### 模式 2：专项优化
触发："镜头抖动怎么解决" / "Boss 战不够刺激"
→ 定位到对应模块，给出具体技巧

### 模式 3：诊断修复
触发："玩家说操作不流畅" / "关卡太单调"
→ 从玩家反馈反推设计问题，定位修复方案

### 模式 4：设计评审
触发："帮我 review 这个关卡设计"
→ 用 236 个技巧逐项检查
